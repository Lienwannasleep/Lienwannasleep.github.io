Hexo常用指令筆記

發表於 2023-07-23 更新於 2023-07-27

本篇基於Hexo預設文章修改而成。

建立一篇新文章

建立新的MD檔案，以My New Post作為檔案名稱。

More info: Writing

運行伺服器

使用主機作為伺服器，產生網頁連結。

More info: Server

產生靜態網頁檔案

產生的檔案會在主資料夾下的public資料夾，將public資料夾的檔案更新至github上即可發布網頁。

More info: Generating

部署遠端網頁

修改_config.yml內，關於deploy的程式碼，將部署網頁連結設定完成，即可透過此指令直接更新程式碼，不需透過github desktop更新資料。

More info: Deployment
